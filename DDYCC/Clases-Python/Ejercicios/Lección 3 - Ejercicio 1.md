## Solución
```
# --- No cambiar el codigo de abajo --- #
number = int(input("Que numero quieres checar? "))
# --- No cambiar el codigo de arriba --- #

####################################
# --- Escribe tu codigo abajo --- #
if number % 2 == 0:
    print("\nEs un numero par.")
else:
    print("\nEs un numero impar.")
```
